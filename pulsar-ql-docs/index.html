Usage Examples

Basic Web Scraping

Extract title and links from a web page:

SELECT
    DOM.text(DOM.selectFirst(dom, 'title')) as title,
    DOM.hrefs(dom) as links
FROM (
    SELECT DOM.load('https://example.com') as dom
) t;

String Processing

Process and validate text data:

SELECT
    STR.capitalize(STR.trim(text)) as clean_text,
    STR.isNumeric(price) as is_valid_price
FROM product_data;

Data Extraction with Regex

Extract structured data using regular expressions:

SELECT
    re1(content, '\$([0-9.]+)') as price,
    re1(content, '([0-9]{4}-[0-9]{2}-[0-9]{2})') as date
FROM articles;

Array Operations

Work with arrays in SQL queries:

SELECT
    ARRAY.joinToString(tags, ', ') as tag_list,
    ARRAY.firstNotEmpty(categories) as primary_category
FROM products;
